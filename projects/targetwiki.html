LG Chem AX Technology Development Team · Apr. – Nov. 2023

TargetWiki: Target Validation Report Auto-Generation Tool

약 20개 외부 DB를 자동 크롤링해 GOT-IT 타겟 검증 리포트를 python-pptx로 자동 생성하는 툴. 비기술직 연구 환경을 위해 단일 클릭 exe로 패키징. 리서치 시간 87% 단축 (120분 → 5~20분).

Skills & Technologies

Python BeautifulSoup Selenium python-pptx PyInstaller tkinter XPath Web Crawling

Problem

신약 타겟 검증 리서치 시 연구원들이 약 20개 이상의 외부 DB(UniProt, GeneCards, ChEMBL 등)에 개별 접속하여 동일/유사 키워드를 반복적으로 입력하고 수집한 정보를 일일이 캡쳐 및 정리하여 수작업으로 리포트를 작성해야 했습니다. 건당 약 120분이 소요되었으며, 리서치 업무의 효율성 및 생산성 향상을 위한 자동화 프로그램 개발에 대한 강한 니즈가 있었습니다.

Solution

크롤링 시스템: BeautifulSoup과 Selenium을 활용하여 각 사이트별 크롤링 및 주요 데이터 로컬 내 저장 기능 구현. 사이트별 XPath 매핑, 오타 자동 교정, 팝업·쿠키 자동 처리 기능을 갖춘 강건한 크롤러 개발.

속도 최적화: 병렬 처리를 적용하여 프로그램 실행 속도를 대폭 최적화. 수집된 데이터는 python-pptx를 활용하여 콘텐츠 양에 무관하게 레이아웃이 동적으로 조정되는 표준화 GOT-IT 검증 리포트로 자동 변환됩니다.

사용 편의성: 비기술 연구 환경을 위해 설치 과정을 제거하고 PyInstaller + tkinter GUI로 패키징하여 단일 클릭 exe 파일로 End-to-End 실행을 지원합니다.

Key Achievements

• 타겟 검증 리서치 시간 87% 단축 (120분 → 5~20분)
• 표준화 리포트 포맷으로 연구자간 품질 편차 제거
• 설치 불필요 단일 exe로 전 연구자 즉시 배포
