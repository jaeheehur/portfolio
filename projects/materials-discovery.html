Target Compound Literature Screening Platform

신소재 발굴을 위한 문헌 인텔리전스 파이프라인. 특허 DB·학술 논문에서 키워드 기반 크롤링 및 API 수집을 수행하고, LLM이 물성·합성 조건·실험 방법을 구조화 테이블로 추출합니다. 연구자가 자연어로 데이터베이스를 조회할 수 있는 Text2SQL 챗봇을 함께 제공합니다.

Skills & Technologies

BeautifulSoup / Selenium LLM Langchain Text2SQL Data Extraction SQL

Overview

연구자가 신소재 후보를 탐색하기 위해 특허와 논문을 수동으로 검색·정리하는 데 소요되는 많은 시간을 단축시키고자 LLM이 생성한 확장 특허 검색 쿼리를 활용해 키워드 기반 검색의 커버리지를 높이고, 수집된 문헌에서 물성, 합성 조건, 실험 방법 등 핵심 정보를 자동 추출하여 구조화된 DB로 적재합니다.

결과 DB에 자연어로 질의할 수 있는 Text2SQL 챗봇을 함께 제공해 연구자가 SQL 없이도 "100도 이상 온도 범위에서 합성된 분자량 200 이상의 화합물"과 같은 쿼리를 직접 실행할 수 있도록 했습니다.

Key Achievements
