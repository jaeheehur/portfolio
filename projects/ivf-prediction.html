LG Chem AX Technology Development Team · Apr. – Jun. 2025

IVF Live Birth Outcome Prediction Model

HFEA(Human Fertilisation and Embryology Authority) 데이터셋을 활용해 IVF 시술의 생존 출생 결과를 예측하는 모델. 이 모델을 기반으로 LG AI Research와 협력해 LG Aimers 6회 해커톤(2,000+ 참가자, 27개 결선팀)을 출제했으며, FAME 2025에서 150+ 의사 참석자 대상으로 발표했습니다.

Skills & Technologies

Python Scikit-learn Feature Engineering LightGBM Pandas Clinical Data Analysis HFEA Dataset

Overview

HFEA 공개 데이터셋의 임상·실험실 특성(시술 유형, 배아 정보, 환자 연령 등)을 기반으로 IVF 생존 출생 결과를 예측하는 분류 모델을 개발했습니다.

LG AI Research와 협력해 이 모델을 LG Aimers 6회 해커톤의 기반 문제로 출제했습니다. 결선 27팀의 특성 엔지니어링 접근법을 검토해 모델 개선에 반영하고, FAME 2025(Fertility Advancement Meeting) 학술회의에서 150명 이상의 의사를 대상으로 발표했습니다.

Key Achievements

• LG Aimers 6회 해커톤 출제 및 심사위원으로 참여 (참가자 2,000+, 결선 27팀)
• FAME 2025 학술회의 발표 (150+ 의사 참석자)
• 해커톤 결선팀 접근법을 반영한 모델 고도화
