ML Project · 2023

Customer Churn Prediction

고객 행동 데이터를 기반으로 이탈 가능성이 높은 고객을 사전에 식별하는 예측 모델. XGBoost와 SHAP을 활용해 높은 예측 정확도와 함께 모델 해석 가능성을 확보했습니다.

Skills & Technologies

Python XGBoost SHAP Scikit-learn Pandas Optuna Matplotlib

Overview

구독 서비스의 고객 이탈을 예측하기 위해 로그인 빈도, 기능 사용 패턴, 결제 이력 등 다양한 행동 데이터를 특성으로 엔지니어링하고 XGBoost 분류 모델을 학습시켰습니다.

Optuna로 하이퍼파라미터 최적화를 수행하고, SHAP으로 개별 예측에 대한 기여도를 시각화해 비즈니스 팀이 결과를 이해하고 액션을 취할 수 있도록 했습니다.

Key Achievements

• AUC-ROC 0.91 달성
• 상위 20% 이탈 위험 고객 대상 리텐션 캠페인 연동
• SHAP waterfall/beeswarm 차트로 모델 해석 리포트 자동 생성
