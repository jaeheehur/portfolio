Featured Project · 2024

Marketing Analytics Dashboard

Google Ads, Meta, Kakao 등 다양한 광고 채널의 성과 데이터를 자동 수집하고, BigQuery에 적재한 뒤 Looker Studio로 실시간 시각화하는 엔드투엔드 데이터 파이프라인 프로젝트입니다. 마케터가 직접 데이터를 조회하고 인사이트를 도출할 수 있도록 셀프서비스 환경을 구축했습니다.

Skills & Technologies

Python Apache Airflow BigQuery dbt Looker Studio Google Ads API Meta Marketing API GCP

Background

마케팅 팀이 매일 아침 수동으로 각 광고 플랫폼에 접속해 데이터를 엑셀로 정리하는 데 1~2시간을 소비하고 있었습니다. 채널 간 성과 비교도 어려워 의사결정이 지연되는 문제가 있었습니다.

Airflow를 활용해 매일 새벽 자동으로 각 채널 API를 호출하고 BigQuery에 적재, dbt로 변환·집계한 뒤 Looker Studio 대시보드에서 채널별/기간별 성과를 한눈에 볼 수 있도록 구성했습니다.
