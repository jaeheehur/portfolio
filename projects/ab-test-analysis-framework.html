A/B Test Analysis Framework

실험 설계부터 결과 분석, 리포트 생성까지 자동화한 A/B 테스트 툴킷. Streamlit 기반 UI로 데이터 분석 지식 없이도 누구나 실험 결과를 해석할 수 있도록 만들었습니다.

Skills & Technologies

Python SciPy Streamlit Pandas Plotly statsmodels

Overview

마케팅·프로덕트 팀에서 A/B 테스트 결과를 매번 데이터팀에 의뢰해야 해서 병목이 발생했습니다. t-test, chi-square, Mann-Whitney 등 다양한 검정 방법을 자동으로 선택하고 적절한 샘플 사이즈 계산, 신뢰구간 시각화까지 지원하는 Streamlit 앱을 만들었습니다.

실험 데이터를 CSV로 업로드하면 유의성 검정 결과와 함께 인터랙티브 차트가 생성되고, 요약 리포트를 PDF로 다운로드할 수 있습니다.

Key Achievements

• 비개발자도 독립적으로 실험 분석 가능하도록 UI 구성
• 데이터팀 분석 요청 건수 약 40% 감소
• 여러 검정 방법 자동 선택 및 결과 해석 가이드 내장
• 샘플 사이즈 계산기 및 파워 분석 기능 포함
