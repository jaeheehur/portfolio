Featured Project · 2024

LLM-based Document Q&A System

사내 문서(PDF, DOCX, 사내 위키 등)를 인덱싱하여 자연어로 질의응답할 수 있는 RAG(Retrieval-Augmented Generation) 파이프라인을 설계하고 구현한 프로젝트입니다. 기존에 문서 검색에 걸리던 시간을 대폭 줄이고, 정확한 출처와 함께 답변을 제공합니다.

Live Demo ↗ GitHub ↗
🤖

Skills & Technologies

Python LangChain OpenAI API ChromaDB FastAPI Docker Streamlit AWS S3

Background

사내에 수백 개의 문서가 분산 저장되어 있어 필요한 정보를 찾는 데 평균 15분 이상 소요되는 문제가 있었습니다. 키워드 검색으로는 문맥을 파악하기 어렵고, 담당자에게 직접 물어봐야 하는 경우가 많았습니다.

이를 해결하기 위해 RAG 아키텍처를 기반으로, 문서를 청크 단위로 분리해 벡터 임베딩 후 ChromaDB에 저장하고, 사용자 질문과 유사한 청크를 검색해 LLM에 컨텍스트로 전달하는 파이프라인을 구축했습니다.

Key Achievements

• 문서 검색 시간 평균 15분 → 30초 이내로 단축
• PDF, DOCX, Markdown, HTML 등 다양한 포맷 지원
• 답변에 출처 문서 및 페이지 번호 함께 제공
• Streamlit UI를 통해 비개발자도 손쉽게 사용 가능
