NLP Project · 2023

NER Fine-tuning for Korean Text

한국어 도메인 특화 텍스트에서 인물, 조직, 날짜, 제품명 등 개체명을 자동 추출하는 NER 모델. KoBERT를 파인튜닝하고 직접 레이블링한 데이터셋으로 학습했습니다.

Skills & Technologies

Python PyTorch HuggingFace Transformers KoBERT Label Studio seqeval Weights & Biases

Overview

뉴스 및 사내 보고서에서 주요 개체명을 자동 추출해 검색 인덱싱과 요약에 활용하기 위한 프로젝트입니다. Label Studio로 약 3,000문장의 도메인 특화 데이터를 직접 레이블링하고, KoBERT 기반 토큰 분류 모델을 파인튜닝했습니다.

Weights & Biases로 실험을 추적하고, seqeval로 엔티티 단위 F1 스코어를 평가했습니다.

Key Achievements

• 도메인 특화 NER F1 Score 0.87 달성
• 6개 엔티티 유형(인물, 조직, 장소, 날짜, 제품, 기타) 인식
• 추론 API 서버로 배포해 하위 파이프라인에서 활용
