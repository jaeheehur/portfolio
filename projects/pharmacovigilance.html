LG Chem AX Technology Development Team · Apr. – Dec. 2024

Pharmacovigilance Literature Monitoring Automation Platform

PubMed 기반 부작용 문헌 감시 자동화 플랫폼. 4-모델 NER 앙상블로 약물·부작용·환자 정보를 추출하고 MedDRA 98% 커버리지를 달성. 약물 감시 규제 보고 의무를 지원합니다.

Skills & Technologies

Python NER (Named Entity Recognition) scispaCy BC5CDR BioBERT ADE-Corpus MedDRA LLT Ensemble PubMed API Streamlit SQL

Problem

강화된 약물 감시 규정으로 인해 문헌 기반의 이상 사례 모니터링 업무의 중요성이 크게 증가했습니다. 그러나 기존 수작업 기반의 문헌 검색 및 이상 사례 정보 수집 방식으로 인해 과도한 시간이 소요되고 업무 비효율이 발생했습니다.

Solution

NER 파이프라인: Spacy 기반 통합 NER 파이프라인 개발 (BC5CDR, ADE Corpus, 자체 구축 Tagging 데이터셋). PubMed API 연동 및 Entity 자동 하이라이팅 기능 구현. Rule 기반 스크리닝 로직 보완으로 키워드 탐지 정밀도 향상.

모델 상세: scispaCy BC5CDR으로 화학물질·질환 개체명 추출, BioBERT ADE-Corpus로 부작용 이벤트 특화 추출, MedDRA LLT Mapper로 추출 개체를 MedDRA 코드로 매핑, Demographics Extractor로 환자 정보(성별, 연령 등) 추출합니다.

LLM 트렌드 분석: LangChain 기반 LLM 파이프라인으로 최신 문헌 트렌드 요약 및 추천 프롬프트 제공. 추출된 데이터를 Dash 기반 웹 대시보드로 시각화하여 검색 결과 및 트렌드 리포트를 제공합니다.

Trend Analysis & LLM Summary

단순 문헌 수집·분류를 넘어, NER 모델과 LLM을 결합해 부작용 트렌드 분석 기능을 구현했습니다. 추출된 개체 데이터를 기반으로 LangChain 기반 LLM 파이프라인이 부작용 발생 패턴의 변화, 신규 안전 시그널, 주요 키워드를 자동으로 식별하고 요약합니다. 이를 통해 규제 담당자가 대량의 문헌에서 핵심 안전 트렌드를 빠르게 파악할 수 있습니다.

Key Achievements

• MedDRA LLT 기준 98% 부작용 커버리지 달성
• NER + LLM 결합으로 부작용 트렌드 자동 분석 및 안전 시그널 식별
• 규제 보고용 문헌 스크리닝 워크플로우 자동화
• 실시간 PubMed 검색 결과 NER 추론 및 시각화
