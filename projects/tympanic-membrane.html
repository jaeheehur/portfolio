Seoul Asan Hospital · Jan. 2019 – Nov. 2020 · Published in PLOS ONE (2022)

Tympanic Membrane Endoscopy Image Classification

고막 내시경 이미지의 병변을 자동 분류하는 앙상블 딥러닝 모델. VGG16, ResNet, DenseNet, EfficientNet 앙상블로 95.8% 정확도를 달성하고 Grad-CAM 활성화 맵으로 임상가 대상 검증 UI를 설계. PLOS ONE 게재 (Choi et al., 2022).

Skills & Technologies

Python PyTorch Keras VGG16 ResNet DenseNet EfficientNet Grad-CAM Ensemble Medical Imaging

Overview

서울아산병원 이비인후과와 협력해 고막 내시경 이미지를 정상, 중이염, 천공 등 다중 병변으로 자동 분류하는 딥러닝 모델을 개발했습니다.

VGG16, ResNet, DenseNet, EfficientNet 4개 모델의 앙상블로 95.8% 분류 정확도를 달성했습니다. Grad-CAM 활성화 맵을 활용해 모델이 어떤 이미지 영역에 근거해 예측했는지를 시각화하는 임상가 대상 검증 UI를 설계해 신뢰도를 높였습니다.

Publication

Choi Y., Chae J., Park K., Hur J., Kweon J., Ahn J. (2022). Automated multi-class classification for prediction of tympanic membrane changes with deep learning models. PLOS ONE. [SCIE]

Key Achievements

• 다중 분류 정확도 95.8% 달성 (4-모델 앙상블)
• Grad-CAM 기반 임상 해석 가능성 UI 설계
• PLOS ONE 게재 (SCIE)
